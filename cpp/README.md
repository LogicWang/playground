### run app code
clang++ -fdiagnostics-color=always -std=c++11 -g ./{app_name}.cpp -o ./out/{app_name}.out



==
compile example
g++ -o hello hello.cpp
or
```bash
gcc hello.cpp -lstdc++ -o hello.out
```

compile with multi source files
```bash
gcc -o main main.cpp hello.cpp -lstdc++
```
run example
./hello
